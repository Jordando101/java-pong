Pong
=========

Java implementation of Pong

Controls
========

Left Player Movement:

+ Up - W key
+ Down - S key

Right Player Movement:

+ Up - Up arrow key
+ Down - Down arrow key

Other:

+ Hit the spacebar to start the game and to continue play after each score.
